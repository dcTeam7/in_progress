# Export Data

Componente que ao ser clicado permite o usuário **exportar** os dados coletados em um arquivo ```CSV```

Ao clicar no botão, será realizado um download de um arquivo onde ira contér os dados de cada aluno.

    }}>

   </Story>

## Propriedades para inicializar o componente

- Você precisa definir seu texto atravéz da propriedade ```text``` essa propriedade é **obrigatória**

```.tsx
<ExportData text="Meu texto"/>
```

O texto será renderizado na tela ao lado de seu ícone

---

- Definir o onClick para utilizar **Javascript** atravéz do seu click.

```.tsx
<ExportData text="Exportar Dados"/>
```
Método ```onClick()``` por default não é **obrigatório** no momento de implementação é possível você ja chamar o componente para que so depois se preocupe de fato em como **implementar a funcionalidade**.

---

## Aplicando o método onClick()

Realizando a implementação do ```onClick()``` é possivel passar as funcionalidades como um ```alert()``` ou qualquer código em Javascript.

<ExportData text="Meu texto" htmlFor="Botão" onClick={() => alert('funcionou')}/>

```.tsx
<ExportData text="Meu texto" htmlFor="Botão" onClick={() => {
  alert('Funcionou')
}}/>
```
